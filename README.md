# Baccara prediction model
## Requirements
- Python 3.7.x;
- TensorFlow 2.3.1. 

## Installation and usage
-   Create virtual env
    ```shell
    python3 -m venv env
    ```
    and activate
    ```shell
    source venv/bin/activate
    ```
-   Install dependencies
    ```shell
    pip install -r requirements.txt
    ```
-   Train model
    ```shell
    python train.py
    ```
